# Gestion Locative & Automobile Pro

Application professionnelle pour la gestion immobilière (locataires, quittances, bilans) et la gestion de flotte automobile (locations, ventes).

## Déploiement
Cette application est configurée pour un déploiement automatique sur Vercel via GitHub.
